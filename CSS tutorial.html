CSS Image editing

As you may notice, My image does not have sharp edges, even though the file does. This is due to the size and border settings of CSS. Inside of your "head" section, put "<style>". Underneath put "img" text with curved brackets besides it. inside of the curved brackets put "width:" text, and then a number of your choice. Without spaces, put "px" next to your number to stand as the pixel count. For example 100px. That will set the size of your image and you can always use chrome source to get your numbers. Now for the curved borders, you stay inside the curved brackets, but make another line. Insted of putting width as your text, simply put "border-radius" and set that to a number of choice, still including the px behind it. if you want a circle, put the number for your border as half the number for your size.

Custom rules with CSS

now that we've covered size and image borders, let's go over custom code rules. You probably noticed the text was a different color, and that is due to my custom rule "hl" make a double space under your image settings and create any 2 letter text. After that, put another set of curved brackets. Inside of those brackets, put text that says "color:" and then choose from red, green, or blue! under that text (still in brackets,) put "font-style" and set it to normal. That is the end of code rule, so now if you put your two letter text in angles, as long as it's surrounding text it will transform that text into the color you chose. After you have completed that, make another line and put</style> there you have it. A code rule, and a fancy new image style.
